# Data Sources

This folder contains small datasets used for teaching purposes in the lecture.

The datasets are standard examples commonly used in data visualization,
machine learning, and network analysis.

## Dataset Overview

- **IRIS-Dataset.csv**  
  Classic Iris dataset, originally from the UCI Machine Learning Repository.

- **lesmiserables.json**  
  Character co-occurrence network from *Les Misérables*, widely used in graph visualization examples.

- **celegans.json**  
  Neural network of *C. elegans*, commonly used in network science.

- **baby_names.csv**  
  Based on publicly available baby name statistics from the U.S. Social Security Administration.
The dataset has been aggregated across years and extended with derived quantities
(e.g., relative proportions) to support visualization tasks such as StreamGraphs.

- **photoviz_dynamic.gexf**  
  Example graph dataset from the Gephi ecosystem.

- **cars.json**  
  Derived from the Car Evaluation dataset (UCI Machine Learning Repository).
The original dataset was transformed into JSON format and slightly preprocessed
to simplify its use in JavaScript and Python-based visualization examples..

- **INED-DemographicData.json**
  Based on demographic data provided by the Institut national d'études démographiques (INED).  
  The dataset has been adapted and possibly aggregated for use in teaching and visualization examples.

- **Gapminder Foundation (https://www.gapminder.org)**
This project uses multiple indicators from the Gapminder dataset
(https://www.gapminder.org/), including GDP per capita and life expectancy
(female, male, and total). Gapminder compiles data from sources such as
the World Bank and the Maddison Project.
Included indicators:
    - GDP per capita
    - Life expectancy (female, male, total)

- **Dataset for Hierarchies**
The file "flare.json" is a synthetic dataset used in D3.js examples
for demonstrating hierarchical visualizations.

- **import-85 dataset**
Schlimmer, J. (1985). Automobile [Dataset]. UCI Machine Learning Repository. https://doi.org/10.24432/C5B01C.

- **titanic.csv**
This dataset was downloaded from the GitHub repository [Data Science Dojo](https://github.com/datasciencedojo/datasets/blob/master/titanic.csv). It corresponds to the dataset 
*Titanic: Machine Learning from Disaster* from Kaggle (2012)
## Notes

- All datasets are small, non-personal, and used exclusively for academic teaching.
- They are widely distributed in the data science and visualization community.
- If any dataset has specific licensing requirements, it is used here in accordance with its original intent for open 
educational use.
