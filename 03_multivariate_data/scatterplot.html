Scatter Plot

A scatter plot is a graph representing pairs of values commonly in Cartesian coordinates. The pattern from plotting a data set might reveal any correlation between the measured variables. Scatter plots are appropriate for visualizing the association/correlation between two quantitative variables.

The scatter plot below provides some interaction. The datasets used in this example are the auto-mpg used in 1983 for statistical purposes and the Iris flower data set introduced in 1936 by Roland Fischer for use in taxonomy problems.
