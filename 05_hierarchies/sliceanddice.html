Slice and Dice Treemap

A treemap is a visualization technique based on containment that partitions the space recursively according to the size of the subtrees. It is space-filling, where leaf nodes are clearly visualized.

Shneiderman's Slice and Dice Algorithm

  • First approach by Shneiderman, 1991
  • Shneiderman's algorithm uses a slice- and-dice technique to subdivide rectangles
  • separation into rectangles by recursively flipping between horizontal and vertical lines, e.g. horizontal for odd hierarchy depths and vertical for even hierarchy depths
  • size metric for box sizes is required
  • categorical metric for box color, e.g. file sizes and file types
  • drawback: many thin and elongated rectangles
